## Note About 

This directory contains the experiment results. 

One one hand, to gather statistical data, we rely on the logs generated by the batch processing system. However, due to the presence of a significant amount of unstructured information containing personal identifiers (such as system information related to SSH communication in the batch processing system).

On the other hand, the generated result files, namely the counter example files, can be far too large. Taking intel009 for example, one counter example file of it with 31400 cycles is 173MB. The result files sum up to be over 30GB.

Due to these reasons, we only provide the final aggregated statistical results in this section.
